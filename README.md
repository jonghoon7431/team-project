# :game_die: 보드메이트(Board Mate)


#### 보드게임 참여 인원을 구할 수 있는 지도 커뮤니티


<br>

## :exclamation: 기능
<br>


- 로그인, 회원가입
- 글 작성, 조회, 수정, 삭제
- 댓글기능
- 마이페이지 - 프로필사진 및 최애 보드게임 설정, 본인 게시글 리스팅
<hr>

  - 게시글 작성, 수정시의 위치 지정 : kakao map api 사용
  
  - db: supabase
